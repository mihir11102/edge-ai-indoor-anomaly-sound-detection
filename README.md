# Edge AI Indoor Anomaly Sound Detection

## Project Overview
This project detects abnormal indoor sounds using machine learning and edge AI. Audio signals are processed using MFCC feature extraction and classified using machine learning models. The final model will be deployed on an ESP-SENSE microcontroller for real-time sound detection.

## Repository Structure

data/  
Contains the dataset used for training and testing.

notebooks/  
Contains Jupyter notebooks used for feature extraction and model training.

src/  
Python scripts for preprocessing and feature extraction.

models/  
Saved trained machine learning models.

deployment/  
Files related to microcontroller deployment.

docs/  
Project documentation and weekly reflections.

reports/  
Final project report and presentation.

## Workflow

Audio → MFCC Feature Extraction → Machine Learning Model → Edge Deployment
